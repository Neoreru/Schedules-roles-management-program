## 기능

### 방 생성 및 입장 기능

* 사용자가 새로운 방을 생성하고 방 이름을 설정하는 기능
* 생성된 방 코드를 통해 다른 팀원이 같은 방에 입장하는 기능
* 한 번 입장한 방 목록을 저장하고, 이후 다시 해당 방에 쉽게 입장하는 기능
* 사용자가 방에서 나가면 자신의 정보가 삭제되는 기능

### 팀원 정보 관리 기능

* 각 사용자가 자신의 이름을 입력하고 수정하는 기능
* 다른 팀원의 정보는 조회만 가능하고, 수정할 수 없도록 제한하는 기능
* 팀원별 가능한 요일과 시간을 시간표에서 선택하거나 해제하는 기능
* 각 팀원의 가능 시간을 시간표 형태로 확인하는 기능

### 공통 시간 확인 기능

* 팀원들이 선택한 가능 시간을 바탕으로 공통 가능한 시간을 계산하는 기능
* 최소 인원을 설정하여 해당 인원 이상이 가능한 시간만 확인하는 기능
* 반드시 포함해야 하는 팀원을 선택하여 조건에 맞는 공통 시간을 찾는 기능
* 공통 가능한 시간을 목록 형태 또는 시간표 형태로 확인하는 기능

### 역할 및 메모 관리 기능

* 각 사용자가 자신의 역할을 입력하고 수정하는 기능
* 팀원별 역할을 확인하는 기능
* 각 사용자가 자신의 메모를 작성하고 수정하는 기능
* 팀원별 역할과 메모를 함께 확인하는 기능

### 데이터 저장 기능

* 팀원 이름, 가능 시간, 역할, 메모를 Firebase에 저장하는 기능
* 새로고침하거나 다른 기기에서 접속해도 저장된 데이터를 불러오는 기능
* 같은 방에 입장한 사용자들이 실시간으로 변경된 정보를 확인할 수 있는 기능

## 페이지 구성

### 시작 페이지

* 방 생성, 방 입장, 입장한 방 목록 확인 기능 제공

### 메인 페이지

* 공통 가능한 시간 확인
* 최소 인원 설정
* 포함해야 하는 팀원 선택
* 공통 시간을 목록 또는 시간표 형태로 표시

### 시간 관리 페이지

* 팀원별 가능 시간 확인
* 자신의 가능 시간 선택 및 수정
* 다른 팀원의 가능 시간 조회

### 역할 관리 페이지

* 팀원별 역할 확인
* 자신의 역할 입력 및 수정
* 자신의 메모 입력 및 수정

## 기술 스택

* 프론트엔드: React
* 데이터베이스 및 백엔드: Firebase Firestore
* 배포: Vercel
